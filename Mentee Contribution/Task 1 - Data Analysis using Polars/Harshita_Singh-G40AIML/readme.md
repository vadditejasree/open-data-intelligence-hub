# Global Ocean Temperature Analytics

An interactive data analytics dashboard built using Python and Streamlit to analyze global ocean temperature trends.

 Features
- Temperature trend visualization
- Country/region-wise analysis
- Interactive charts
- Climate insights

 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

Link-https://globaloceantemperatureanalytics75.streamlit.app

```bash
pip install -r requirements.txt
streamlit run dashboard.py
