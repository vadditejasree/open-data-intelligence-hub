# Extinction Trend Analytics

## Overview

Extinction Trend Analytics is a data analytics dashboard built using Python and Streamlit. The project analyzes species extinction data and visualizes conservation trends through charts and tables.

## Features

* Dataset preview
* Species category analysis
* Population trend analysis
* Bar charts and pie charts
* Search species by name
* Dataset statistics
* Top and bottom records display
* Interactive Streamlit dashboard

## Technologies Used

* Python
* Pandas
* Streamlit
* Plotly

## Installation

1. Clone the repository

git clone https://github.com/shadik102/Extinction-Trend-Analytics.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py
